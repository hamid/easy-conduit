# آموزش نصب conduit در سه قدم ساده، بدون نیاز به نوشتن کد و یا دانش سرور

اگر وارد این لینک شدی احتمالا میدونی Conduit چیه. پس اگر میدونی قسمت مقدمه رو بی خیال بشو و برو سراغ مرحله اول.

## مقدمه: Conduit چیه و چطور کمکمون می‌کنه؟

Psiphon Conduit یک ابزار قدرتمند برای کمک به دسترسی آزاد به اینترنت است که به شما امکان می‌دهد یک سرور پروکسی برای دور زدن سانسور راه‌اندازی کنید.

## مرحله اول:‌ ثبت نام در دیتاسنتر

برای سرور conduit شما می‌تونید از هر جایی خواستید سرور بگیرید. دیتاسنتر زیر رو که من معرفی کردم دو تا خوبی داره:

- اول اینکه فقط با اضافه کردن کارت بانکیتون می‌تونید سریع سرور بسازید و نیاز به تایید هویت و ارسال مدارک اضاف نیست
- دوم اینکه چند خط کد لازم رو توی یک محیط گرافیکی اضافه میکنید و بعد از اون هر چقدر سرور ساختید اون کد ها اتوماتیک اجرا می‌شن و نیاز به کار دیگه ای نیست

### دستور نصب سریع

```bash
wget https://raw.githubusercontent.com/YOUR_USERNAME/conduit/master/conduit-start-script.sh
chmod +x conduit-start-script.sh
sudo bash conduit-start-script.sh
```
